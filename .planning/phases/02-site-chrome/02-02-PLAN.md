## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-03 | Tampering | accountUrl / nav-link url props used as href | accept | React escapes attribute values; platform sanitizes URLs before delivery; static bundle has no server-side risk. javascript: URLs are low-risk for a static render — platform responsibility per RESEARCH.md Security Domain. |
| T-02-04 | Tampering | logo image src via ImageGuard | accept | ImageGuard renders src as img src attribute — React escapes; platform-uploaded images go through the media library; no user-controlled raw URL injection in standard flow. |
| T-02-05 | Information Disclosure | SiteHeader renders accountUrl as visible href in DOM | accept | accountUrl is intentionally public navigation — no sensitive data; this is standard nav link behavior. |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | Zero new package installs this plan (RESEARCH.md Package Legitimacy Audit: no new dependencies). |
</threat_model>
